# EasyCharts
A simple chart lib used for iOS

## Requirements
* ARC
* iOS>=8.0
* iPhone  || iPad

## Features

EasyCharts supports multiple methods for installing the library in a project.

## Installation

* using CocoaPods
* using Carthage
* by cloning the project into your repository
